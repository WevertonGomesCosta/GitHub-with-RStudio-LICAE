Pré requisitos para o Curso Integrando GitHub com o RStudio

1- R version > 4.1: https://cran.r-project.org/

2- RStudio version > RStudio Desktop 2022.07: https://www.rstudio.com/products/rstudio/download/#download

3- Install packages - tidyverse (includes dplyr, tidyr, ggplot2, magrittr and other really useful ones) - workflowr

4- Criar conta no GitHub: https://github.com/

É possível ter uma conta de estudantes no github, com acesso à vários beneficios: https://education.github.com/students

5- Instalar o Git Bash: Git BASH for Windows
